Get-Parameters
-filename pathtoparameterfile
Reads a parameter file and returns a List<Parameter>

-logfile pathtologfile
Reads a log file and parses out the parameters used in the log. Returns a List<Parameter>

Set-Parameter
-parameter parametername -value newvalue
Given a list of parameters from pipeline or using -parameters List<Parameter>, set a new value for a parameter

-EnvironmentVariables
Given a list of parameters from pipeline or using -parameters List<Parameter>, replaces any environment variables (indicated by %name%) with their values

Start-AxSetup
-setupPath pathandfilenameofsetupexe
Given a list of parameters from pipeline, using -parameters List<Parameter> or by specifying -filename pathandfilenameofparametersfile, start AX setup and pass the parameters as a file
When passed a list of parameters, the parameters will be saved to file in %temp% with a GUID as a filename.
Note that the cmdlet will wait indefinitely for the setup process to exit.
