Server

Get-AOSNames
Reads the registry and returns a List<string>, containing the names of he AOSes

Get-AOSIDs
Reads the registry and returns a List<uint>, containing the names of he AOSes

Get-ServerConfiguration
-listnames {-aosname name | -aosnumber id}
Reads the registry for server configurations for the given aos name or id and returns a List<string> containing the configuration names

-active {-aosname name | -aosnumber id}
Reads the registry and retrieves the active configuration for the given aos name or id and returns a Server object containing the configuration

-config configurationname {-aosname name | -aosnumber id}
Reads the registry and retrieves the specified configuration for the given aos name or id and returns a Server object containing the configuration

-filename configurationfilename
Reads a configuration from a server configuration (.axc) file returns a Server object containing the configuration

Set-ServerConfiguration
-config configurationname {-aosname name | -aosnumber id}
Specifies a name of a configuration to make active for the given aos name or id

Save-ServerConfiguration
-filename configurationfilename
Given a Server object containing a configuration (from pipeline or using -ServerConfiguration), write the configuration to a server configuration file (.axc)

{-aosname name | -aosnumber id} [-config configurationname]
Given a Server object containing a configuration (from pipeline or using -ServerConfiguration), write the configuration to the registry for the given aos name or id.
Note that if a configuration name is not specified using -config, the configuration name contained in the Server object will be used.


Client

Get-ClientConfiguration
-listnames [-BC]
Reads the registry for client configurations for the current user, or for the business connector when specifying -BC, and returns a List<string> containing the configuration names

-active [-BC]
Reads the registry and retrieves the active configuration for the current user, or for the business connector when specifying -BC, and returns a Client object containing the configuration

-config configurationname {-aosname name | -aosnumber id}
Reads the registry and retrieves the specified configuration for the current user, or for the business connector when specifying -BC, and returns a Client object containing the configuration

-filename configurationfilename
Reads a configuration from a client configuration (.axc) file returns a Client object containing the configuration

Set-ClientConfiguration
-config configurationname [-BC]
Specifies a name of a configuration to make active for the current user, or for the business connector when specifying -BC

Save-ClientConfiguration
-filename configurationfilename
Given a Client object containing a configuration (from pipeline or using -clientConfiguration), write the configuration to a client configuration file (.axc)

[-BC] [-config configurationname]
Given a Client object containing a configuration (from pipeline or using -clientConfiguration), write the configuration to the registry for the current user, or for the business connector when specifying -BC.
Note that if a configuration name is not specified using -config, the configuration name contained in the Client object will be used.
